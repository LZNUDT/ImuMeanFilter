# ImuMeanFilter

## Imu数据均值滤波分析

Imu数据分析中常用均值滤波，均值滤波后的数据可以直观的看出，传感器的零偏稳定性、零偏和温度的关系、传感器数据和时间的关系。虽然得不到具体的数据指标，但是可以的感觉到传感器的好坏，传感器测试非常有必要。
